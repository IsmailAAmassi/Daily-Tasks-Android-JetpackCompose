# Daily Tasks App
## About Daily Tasks
An android application developed using Jetpack Compose 

##Built With
- Kotlin version: 1.6.21
- Jetpack Compose version: 1.2.0-beta03
- Architecture: MVI
- Coroutines
- Coil
- Navigation: raamcosta.compose-destinations
- Hilt
- Datastore
- Kotlin Gradle
